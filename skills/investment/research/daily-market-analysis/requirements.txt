# Daily Market Analysis - Requirements

## Core Dependencies

```
akshare>=1.12.0        # A 股/港股/大宗商品数据
yfinance>=0.2.0        # 美股数据
requests>=2.31.0       # HTTP 请求
python-dateutil>=2.8.0 # 日期处理
pandas>=2.0.0          # 数据处理
```

## Optional Dependencies

```
tavily-python>=0.3.0   # AI 新闻搜索
tradingeconomics       # 经济日历
beautifulsoup4>=4.12.0 # HTML 解析
feedparser>=6.0.0      # RSS 解析
```

## Installation

```bash
# 基础安装（免费数据源）
pip install akshare yfinance requests python-dateutil pandas

# 完整安装（包含付费 API）
pip install akshare yfinance requests python-dateutil pandas tavily-python tradingeconomics beautifulsoup4 feedparser
```

## Python Version

- Python 3.10+
- Python 3.11 recommended

## System Requirements

- Memory: 512MB+
- Disk: 100MB+
- Network: Required for data fetching
